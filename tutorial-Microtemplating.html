Microtemplating

HTML strings are boring and complex, they take a lot of space. Let's fix that with emmet:

  • nav>ul>li instead of
  • form#search.wide instead of
  • [a='value1' b="value2"] instead of
  • ul>li.item$*3 instead of

Because of code size emmet expressions support is only for HTML strings for now. Take a look at the emmet cheat sheet for more examples.
